All the information about S2DE is inside lib/S2DE_pipe.c

To run S2DE_bipipe with the given example program 'techt', use commands :

	#1) compile
	make

	#2) run
	./peer_example ./S2DE_bipipe

More arguments can be given as well in run command, they will be translated to the 'S2DE_bipipe' subprocess.

Enjoy !
